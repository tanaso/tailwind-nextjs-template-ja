# はじめに

Tailwind Starter Blogには、[Next.jsの組み込み画像コンポーネント](https://nextjs.org/docs/api-reference/next/image)のサポートが標準で含まれており、MarkdownやMdxドキュメントでのデフォルトの画像タグを自動的にImageコンポーネントに置き換えます。

# 使い方

新しいページルートやJavaScriptファイルで使用するには、画像コンポーネントをインポートして呼び出すだけです。例：

```js
import Image from 'next/image'

function Home() {
  return (
    <>
      <h1>私のホームページ</h1>
      <Image src="/me.png" alt="著者の写真" width={500} height={500} />
      <p>私のホームページへようこそ！</p>
    </>
  )
}

export default Home
```
