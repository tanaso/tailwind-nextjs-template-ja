## インラインハイライト

インラインハイライトの例：`sum = parseInt(num1) + parseInt(num2)`

## コードブロック

いくつかのJavaScriptコード

```javascript
var num1, num2, sum
num1 = prompt('最初の数字を入力してください')
num2 = prompt('2番目の数字を入力してください')
sum = parseInt(num1) + parseInt(num2) // "+" は「加算」を意味します
alert('合計 = ' + sum) // "+" は文字列として結合を意味します
```
