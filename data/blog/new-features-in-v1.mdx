## テーマカラー

テーマカラーは、Tailwindの設定ファイル内で`primary`属性を変更するだけで簡単に変更できます。

```js:tailwind.config.js
theme: {
    colors: {
      primary: colors.teal,
      gray: colors.neutral,
      ...
    }
  ...
}
```
